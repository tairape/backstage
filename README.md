# backstage
Project with HC-SR04 sensor for installation on backstage of the web series 2.

# List of materials
Here is the list of materials used for building the backstage lights.

- 1x Arduino board + USB cable
- 1x HC-SR04
- Jumpers
- Wires
- 1 enclosure?

# Connection
- Trig goes to PIN 3
- Echho goes to PIN 2
